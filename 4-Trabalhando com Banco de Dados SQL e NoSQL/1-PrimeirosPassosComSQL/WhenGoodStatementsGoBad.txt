Em tecnologias de banco de dados, um rollback é uma operação que retorna o banco de dados a algum estado anterior. As reversões são importantes para a integridade do banco de dados, pois significam que o banco de dados pode ser restaurado para uma cópia limpa mesmo após a execução de operações incorretas.
