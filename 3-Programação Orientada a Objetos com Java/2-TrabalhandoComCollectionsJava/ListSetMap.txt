Collection é um objeto que agrupa múltiplos elementos (Variáveis primitivas ou objtetos) dentro de uma única unidade. Serve para armazenar e processar conjuntos de dados de forma eficiente.

List -- java.util.List
Elementos duplicados e garante ordem de inserção
ArrayList deve ser usada onde mais operações de pesquisa são necessárias, e LinkedList deve ser usada onde mais operações de inclusão e exclusão são necessárias.
