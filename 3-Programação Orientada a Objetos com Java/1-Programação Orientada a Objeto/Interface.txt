Define um contrato que deve ser seguido pela classe que a implementa. Quando uma classe implementa uma interface, ela se compromete a realizar todos os comportamentos que a interface disponibiliza.

Java:
interface A {
	...
}

Class B implements A {
	...
}

C#:
interface A {
	...
}

Class B : A {
	...
}

Python não possúi.

Tipos de classes: Abstrata e Concreta

Palavras coriga para herança: Java: super C#: base python: super()
Relação entre classes e interface: extends e implementes
