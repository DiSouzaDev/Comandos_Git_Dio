Operadores relacionais são símbolos especiais que são capazes de realizar comparações entre determinados operandos.

Utilizados para igualdade ou diferença
Igualdade: ==
Diferença: !=

Operadores lógicos são símbolos especiais que são capazes de realizar comparações lógicas entre operadores lógicos ou expressões e, em seguida, retornar um resultado.
Tipos:
Conjunção: &&
Disjunção: ||
Disjunção exclusiva: ^ -- Quando ambos os operadores são opostos
Negação: ! -- Inverte o valor lógico de um operador
